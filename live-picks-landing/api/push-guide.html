Push notification setup

Turn on iPhone pick alerts.

If Safari says push is unsupported, the site is probably open as a browser tab. On iPhone, Live Picks needs to be added to the Home Screen and opened from that icon before alerts can be enabled.

Three steps.

Follow these on your iPhone. The screenshots are simplified, but the buttons and order are the same.

Step 01

Tap the Share button in Safari.

Open Live Picks in Safari and tap the Share button in the bottom toolbar. This opens the iPhone share sheet.

Start in Safari
Step 02

Choose Add to Home Screen.

Scroll the share sheet if needed, tap Add to Home Screen, confirm the name Live Picks, then tap Add.

Install the web app
Step 03

Log in from the Home Screen icon.

Close Safari, open Live Picks from the new Home Screen icon, log in, go to the Premium Feed, tap Enable push notifications, then choose Allow.

Quick fixes.

Most issues are caused by opening the normal website tab instead of the installed Home Screen app.

Still unsupported Open the Home Screen icon.

Do not enable push from a Safari tab. Launch the installed Live Picks icon and try again there.

No prompt Check notification settings.

Go to Settings, Notifications, Live Picks. Turn Allow Notifications on. If Live Picks is missing, remove the icon and add it again.

Desktop Use the normal feed button.

On desktop, open the premium feed in Chrome, Edge, or Safari and click Enable push notifications.
